K3s Resume Platform

Personal DevSecOps / GitOps Homelab Project

Built a GitOps-driven K3s platform to host a resume application and RSS aggregation service, separating application source from cluster desired state across dedicated repositories.

  • Implemented GitHub Actions CI for build, publish, SBOM generation, vulnerability scanning, and artifact signing
  • Used GHCR, Flux CD, Kustomize, and image automation resources for registry monitoring and deployment reconciliation
  • Added Trivy, Syft, Cosign, and Kyverno to strengthen supply chain visibility and production controls
  • Enforced digest-pinned prod images and blocked mutable image patterns through cluster policy
